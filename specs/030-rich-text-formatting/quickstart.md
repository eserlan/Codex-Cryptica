# Quickstart: Rich Text Controls

## Overview

This feature adds a visual toolbar to the Markdown editor, allowing users to format text without knowing Markdown syntax. It also introduces a "Zen Mode" for distraction-free writing.

## Usage

### Using the Toolbar

1.  Open an entity in the Detail Panel.
2.  Click "Edit" to enter edit mode.
3.  A toolbar will appear above the text area.
4.  Select text and click buttons (Bold, Italic, etc.) to apply formatting.
5.  Active formats are highlighted in green.

### Zen Mode

2.  The editor expands to fill the entire screen.
3.  Click the "Minimize" (Exit Zen Mode) icon to return to the standard view.
4.  Press `Esc` to exit Zen Mode.

### Keyboard Shortcuts

- **Bold**: `Cmd+B` / `Ctrl+B`
- **Italic**: `Cmd+I` / `Ctrl+I`
- **Code**: `Cmd+E` / `Ctrl+E`
- **Zen Mode**: `Cmd+Shift+F` (proposed)
