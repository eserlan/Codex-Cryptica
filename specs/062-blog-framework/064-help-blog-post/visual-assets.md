## Quality Standards

- **Resolution**: Minimum 1920x1080 for static captures.
- **Theme**: Ancient Parchment (Primary).
- **Aspect Ratio**: 16:9 or 4:3 depending on the component context.
- **Annotations**: Use themed callout arrows (ink-style for fantasy) if highlighting specific buttons.
