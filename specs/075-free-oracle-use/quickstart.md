# Quickstart: Free Oracle Use

1. **Test System Path**:
   - Clear your browser's local storage (or remove the AI API Key in settings).
   - Open the Oracle Sidebar.
   - You should see "Advanced Neural Uplink" status.
   - Send a message and verify it works (via the proxy).

2. **Test Sovereign Path**:
   - Go to Settings > AI.
   - Enter a valid Gemini API Key.
   - Return to the Oracle Sidebar.
   - You should see "Secure Uplink: Sovereign Mode" status.
   - Send a message and verify it goes directly to Google (Network tab).
