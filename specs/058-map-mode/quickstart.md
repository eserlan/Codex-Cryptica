# Quickstart: Interactive Campaign Mapping & Spatial Lore

## Initial Map Setup

1. Upload a campaign map image (PNG/JPG).
2. The image appears in the "Map Mode" viewport.
3. Use the mouse scroll wheel to zoom and drag to pan.

## Adding Lore Pins

1. Double-click anywhere on the map to drop a pin.
2. An "Unlinked Pin" dialog appears.
3. Type the name of an existing NPC or Location.
4. The pin now shows the icon associated with that entity category.

## Navigating Hierarchy

1. Open a "World Map".
2. Click a pin for a "Major City".
3. If the City entity has an attached sub-map, an "Enter City" button appears.
4. Clicking it switches the viewport to the City's floor plan.

## Fog of War

1. Toggle "GM Mode".
2. Use the "Fog Brush" to hide an unexplored cave system.
3. Toggle "Player View" to verify the cave is masked by themed textures.
