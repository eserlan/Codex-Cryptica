# Entity Timeline

The Entity Timeline gives each entity its own focused history view. Open any entity's detail panel and click the **Timeline** tab to see every event linked to that entity, sorted from earliest to latest.

## Reading the Timeline

Each row shows:

- **Event title**
- **Date or date range**, formatted to your campaign calendar
- **Category label**, if the event has one
- **Summary**, a short excerpt from the event's notes
- **Participants**, other entities linked to the same event

Click any row to open that event's full detail page.

## Undated Events

Events that have no date assigned still appear in the Timeline, grouped under an **Undated** heading at the bottom. No date is invented or guessed — the event is shown as-is.

## Empty State

If an entity has no linked events yet, the Timeline shows a prompt explaining that. To start building the entity's history, open or create an event and link it to the entity using the entity's connections.

## What Counts as Linked

An event appears in the Timeline when there is a direct connection between the entity and the event — either the entity links to the event, or the event links to the entity. Indirect connections (links of links) are not followed.

The Timeline is read-only. It never creates, edits, or infers lore.
